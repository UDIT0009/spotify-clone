# Spotify Clone 🎵

A front-end Spotify Clone built using HTML, CSS, and JavaScript that replicates core music streaming features such as audio playback and interactive UI.

---

## 🚀 Features
- Music playback functionality
- Interactive and responsive UI
- Play, pause, and track controls
- Dynamic user interface using JavaScript

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## 🎯 Purpose
This project was built to understand how real-world music streaming platforms handle user interaction, UI design, and media playback. It helped me strengthen my front-end development skills and gain hands-on experience in building interactive web applications.

---

## 📂 Project Structure
- `index.html` – Main structure of the application  
- `style.css` – Styling and layout  
- `script.js` – Functionality and logic  
- `assets/` – Images and audio files  

---

## ⚠️ Note
Audio files are not included in this repository to keep the project lightweight and within repository size limits. The project focuses on demonstrating UI, functionality, and playback logic.

You can integrate your own audio files in the assets folder to test playback functionality.
